# Quick Setup Guide

## Installation Steps

1. Open terminal in the project directory:
   ```bash
   cd c:\Users\AnnU\OneDrive\Desktop\demo\fashion-store
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. The website will open automatically at http://localhost:3000

## Tech Stack

- React.js 18
- React Router DOM 6
- Context API for state management
- CSS3 with custom properties
- Google Fonts (Cormorant Garamond + Montserrat)

## File Structure

- `/src/components` - Reusable components (Navbar, Footer, ProductCard)
- `/src/pages` - Page components (Home, ProductListing, ProductDetail, Cart)
- `/src/context` - Global state management
- `/src/data` - Mock product data (15 items)
- `/src/styles` - CSS files for each component

## Key Features

1. **Cart Management**: Add/remove items, update quantities, persists in localStorage
2. **Wishlist**: Save favorite products
3. **Filters**: Category and price range filtering
4. **Sorting**: By price (low/high) and rating
5. **Search**: Real-time product search
6. **Responsive**: Works on mobile, tablet, and desktop
